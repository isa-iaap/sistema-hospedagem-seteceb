# Sistema de Hospedagem SETECEB

Projeto extensionista desenvolvido para auxiliar no controle de hospedagens da instituição SETECEB.

## Sobre o projeto
O sistema tem como objetivo organizar a gestão de quartos e suítes, permitindo o controle de hóspedes, reservas e períodos de hospedagem.

## Funcionalidades (MVP)
- Cadastro de quartos e suítes
- Controle de reservas
- Registro de hóspedes
- Controle de datas de entrada e saída
- Visualização de disponibilidade dos quartos

## Objetivo
Facilitar a organização das hospedagens durante eventos, evitando conflitos e melhorando o controle das acomodações.
